# S3 and SQS Ingestion Patterns

**Local testing**: S3 and SQS run in Docker Compose via Localstack. DAGs use `S3Hook` and `SqsHook` with `aws_conn_id="aws_dag_executor"`; connection points at `http://localstack:4566`.

## Write Parquet to S3

In Docker Compose, S3 is Localstack; use `S3Hook(aws_conn_id="aws_dag_executor")`.

```python
from airflow.providers.amazon.aws.hooks.s3 import S3Hook
from utils.s3_utils import write_parquet_to_s3

s3_hook = S3Hook(aws_conn_id="aws_dag_executor")

write_parquet_to_s3(
    s3_hook=s3_hook,
    df=df,
    bucket=bucket_name,
    prefix=f"{base_prefix}year={year}/month={month:02d}/day={day:02d}/data_{job_id}_",
)
```

## Partition Layout

```
{prefix}year={Y}/month={M:02d}/day={D:02d}/data_{job_id}_{timestamp}.parquet
```

## Read Bronze/Silver Parquet

```python
import tempfile
import pandas as pd

partition_prefix = f"{base_prefix}year={year}/month={month:02d}/day={day:02d}/"
all_keys = s3_hook.list_keys(bucket_name, prefix=partition_prefix)
keys = [k for k in all_keys if k.endswith(".parquet")]

df_list = []
with tempfile.TemporaryDirectory() as temp_dir:
    for key in keys:
        s3_hook.download_file(
            key=key,
            bucket_name=bucket_name,
            local_path=temp_dir,
            preserve_file_name=True,
            use_autogenerated_subdir=False,
        )
        df_list.append(pd.read_parquet(os.path.join(temp_dir, os.path.basename(key))))
df = pd.concat(df_list, ignore_index=True)
```

## SQS Receive and Send

In Docker Compose, SQS is Localstack; use `aws_conn_id="aws_dag_executor"` (connection extra: `{"endpoint_url": "http://localstack:4566"}`).

```python
from airflow.providers.amazon.aws.hooks.sqs import SqsHook
import json

sqs_hook = SqsHook(aws_conn_id="aws_dag_executor")
messages = sqs_hook.receive_messages(queue_url=queue_url, max_number_of_messages=10)

for msg in messages:
    body = json.loads(msg["Body"])
    url = body["url"]
    # process...
    sqs_hook.delete_message(queue_url=queue_url, receipt_handle=msg["ReceiptHandle"])

# Send
sqs_hook.send_message(queue_url=queue_url, message_body=json.dumps({"url": url, "source": "homepage"}))
```
